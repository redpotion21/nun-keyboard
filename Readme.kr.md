# 눈 키보드

[日本語](Readme.md)  |  [English](Readme.en.md)  |  한국어

키보드만으로 소라의 방송 내에서 멤버쉽 스탬프를 입력한다!! 이모지 탭에서 일일히 찾는것 보다 빠름!!

# 설치방법

1.https://www.autohotkey.com/ 에서 AutoHotkey(v2.0이상)을 다운로드.

2.다운로드한 파일을 실행하여 설치.

3.이 페이지에 있는 nun keyboard_kr.ahk를 다운로드하여 실행.

# 사용법

<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/80d6b0aa-18f4-4ae8-a826-88e8f5f8973f" />

1.(L Ctrl + R Ctrl)을 동시에 누르면 스탬프 모드가 활성화된다. 다시 누르면 비활성화된다.

2.스탬프 모드가 켜진 채로 소라의 방송내에서 키를 눌르면, 위 사진의 매핑에 따라 해당되는 이모지가 입력된다!

**3.강제종료는 1초 이상의 간격을 두지 않고 (Esc)키를 4번 누를 것.**
